# Accessible Landing Page UI

A modern and fully responsive landing page built with **HTML** and **SCSS**, designed with a strong focus on **accessibility**, clean architecture, and smooth user interactions.

## Key Features
- Responsive layout for all screen sizes
- Built with SCSS (modular and maintainable structure)
- Dark UI with consistent color system
- Accessibility-friendly design for users with disabilities
- Smooth hover and transition effects
- Clean and readable codebase

## Accessibility
This project includes accessibility considerations such as:
- Proper color contrast
- Readable font sizes
- Clear visual focus states
- User-friendly interactions for diverse users

## Technologies Used
- HTML5
- SCSS (Sass)
- CSS3
